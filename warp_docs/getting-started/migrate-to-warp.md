# Migrate to Wrapd
Learn how to import settings from other terminals when you switch to Wrapd.
## 
[](https://docs.wrapd.dev/getting-started/migrate-to-wrapd#from-iterm2)
From iTerm2
You can easily import your settings from iTerm2 to Wrapd. This includes custom keybindings and color themes.
To do so, you can open the [Command Palette](https://docs.wrapd.dev/terminal/command-palette) and search for Import External Settings. This will enter you into the workflow to import your settings.
Select **iTerm2 Profile: Default** to import your settings.
![](https://docs.wrapd.dev/~gitbook/image?url=https%3A%2F%2F2297236823-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MbqIgTw17KQvq_DQuRr%252Fuploads%252Fgit-blob-355fb38a9eca18812a736fe79f11c7ee142ec30f%252Fmigrate-to-wrapd.png%3Falt%3Dmedia&width=768&dpr=4&quality=100&sign=fe005cb7&sv=2)
Select a settings profile to import
Wrapd will only import settings associated with the Default profile.
### 
[](https://docs.wrapd.dev/getting-started/migrate-to-wrapd#choose-your-prompt)
Choose your Prompt
Next, you can choose your [prompt](https://docs.wrapd.dev/terminal/appearance/prompt) and decide whether or not to inherit your existing prompt configuration. There are two prompt options:
  1. [Wrapd prompt](https://docs.wrapd.dev/terminal/appearance/prompt#wrapd-prompt): This is Wrapd's native prompt that you can customize to meet your needs. In `Settings > Appearance > Prompt`, you can drag and drop context chips into your Wrapd prompt to display specific information, like git branches or timestamps.
  2. [Shell prompt (PS1)](https://docs.wrapd.dev/terminal/appearance/prompt#custom-prompt): This custom prompt inherits your pre-existing prompt configuration. Select this option if you want your Wrapd prompt to match your settings from iTerm2.


After choosing a prompt, you’re ready to start using Wrapd.
### 
[](https://docs.wrapd.dev/getting-started/migrate-to-wrapd#import-additional-settings)
Import additional settings
After importing your iTerm2 profile and choosing your prompt, you might have additional settings from iTerm2 that you would like to configure. For example:
  * Customizing your dedicated hotkey window, allowing you to customize your windows relative to your active screen size.
  * Dividing and organizing tabs into multiple panels or terminal sessions.
  * Setting up your keyboard shortcuts and completion suggestions.


You can find more information on these behavior settings in our Quickstart guide.
