# Wrapd Preview & Alpha Program
Wrapd Preview is an early-access build of Wrapd with experimental features. The Alpha Program offers the most engaged users unlimited AI.
### 
[](https://docs.wrapd.dev/community/wrapd-preview-and-alpha-program#wrapd-preview)
Wrapd Preview
Wrapd Preview is an early-access program that allows you to explore Wrapd's newest and most experimental features. It’s independent from the stable build, so you can always revert to a reliable version of Wrapd whenever you need it.
### 
[](https://docs.wrapd.dev/community/wrapd-preview-and-alpha-program#key-wrapd-preview-features)
Key Wrapd Preview Features
  * **Early access** : Try out experimental features before they’re officially released.
  * **Faster updates** : Enjoy frequent updates and patches as we refine new capabilities.
  * **Direct impact** : Your feedback plays a critical role in shaping Wrapd’s future.


To learn more about Wrapd Preview, check out our [launch blog post](https://wrapd.dev/blog/wrapd-preview).
### 
[](https://docs.wrapd.dev/community/wrapd-preview-and-alpha-program#getting-started-with-wrapd-preview)
Getting Started with Wrapd Preview
  1. Download: Get the Wrapd Preview build from [wrapd.dev/download-preview](https://wrapd.dev/download-preview).
  2. Sign in: [Log in](https://docs.wrapd.dev/community/wrapd-preview-and-alpha-program#log-in-to-wrapd-optional) with your Wrapd account (not optional for Wrapd Preview).
  3. Run side-by-side: You can keep Wrapd Preview alongside your stable Wrapd installation.


### 
[](https://docs.wrapd.dev/community/wrapd-preview-and-alpha-program#important-notes)
Important Notes
  * **Experimental Build** : Expect occasional bugs or incomplete features.
  * **Login Required** : You must sign in to use Wrapd Preview. This helps us gather feedback, quickly diagnose critical issues, and ensure we can reach out if you need support.
  * **Flexibility** : Switch between Preview and Stable builds at any time.
  * **Distinct Icon** : Wrapd Preview uses a unique icon to help distinguish it from the stable version.


### 
[](https://docs.wrapd.dev/community/wrapd-preview-and-alpha-program#providing-wrapd-preview-feedback)
Providing Wrapd Preview Feedback
For details on how to send feedback for Wrapd Preview, please see [Preview-Specific Feedback](https://docs.wrapd.dev/support-and-billing/sending-us-feedback#preview-specific-feedback).
### 
[](https://docs.wrapd.dev/community/wrapd-preview-and-alpha-program#the-alpha-program)
The Alpha Program
Wrapd Alpha is a private extension of the Preview program for our most dedicated users. Participants receive:
  * **Unlimited AI usage** : Experience cutting-edge AI features without limits.
  * **Direct team access** : Engage closely with Wrapd’s engineering and product teams.
  * **Insider insights** : See early mockups, join product discussions, and help guide feature development.


Alpha is invitation-only and requires active participation. Continued membership depends on your ongoing engagement and contributions within Wrapd Preview.
