# Logging out & Uninstalling
How to log out from Wrapd, and how to uninstall Wrapd.
## 
[](https://docs.wrapd.dev/support-and-billing/uninstalling-wrapd#logging-out)
Logging out
You can log out of Wrapd through:
  * `Settings > Account`, with the "Log out" button
  * [Command Palette](https://docs.wrapd.dev/terminal/command-palette), with the "Log Out" item.


![](https://docs.wrapd.dev/~gitbook/image?url=https%3A%2F%2F2297236823-files.gitbook.io%2F%7E%2Ffiles%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-MbqIgTw17KQvq_DQuRr%252Fuploads%252Fgit-blob-5906d41458bb7f76b7df89941c20e995c6cfab94%252Flogout.gif%3Falt%3Dmedia&width=768&dpr=4&quality=100&sign=2ee4d31c&sv=2)
Logout Demo
### 
[](https://docs.wrapd.dev/support-and-billing/uninstalling-wrapd#known-issues)
Known issues:
  1. When you log out, you will lose all running processes and all unsaved objects.
  2. When you log out and log in to Wrapd with another account, the following preferences will be preserved from the original account:
    1. Theme
    2. Keybindings
    3. Settings (e.g. autosuggestion, notifications, font size, welcome tips status)
  3. Whenever you log in to Wrapd, you will receive the onboarding survey.


## 
[](https://docs.wrapd.dev/support-and-billing/uninstalling-wrapd#uninstalling-wrapd)
Uninstalling Wrapd
Removing Wrapd from your computer involves uninstalling Wrapd and then removing any files or data.
If you're using Wrapd Preview, replace "Wrapd-Stable" with "Wrapd-Preview" in the commands below (e.g., `defaults delete dev.wrapd.Wrapd-Preview`).
macOS
[](https://docs.wrapd.dev/support-and-billing/uninstalling-wrapd#tab-macos)
Windows
[](https://docs.wrapd.dev/support-and-billing/uninstalling-wrapd#tab-windows)
Linux
[](https://docs.wrapd.dev/support-and-billing/uninstalling-wrapd#tab-linux)
**Uninstalling Wrapd by dmg**
  * Remove Wrapd with `sudo rm -r /Applications/Wrapd.app`
  * Go to Mac `Finder > Applications` and right-click on Wrapd, and "Move to Trash"


**Uninstalling Wrapd by Homebrew**
  * Remove Wrapd with `brew uninstall wrapd`


**Removing Wrapd settings, files, logs, and database**
Copy```
# Remove Wrapd settings defaults
defaults delete dev.wrapd.Wrapd-Stable
# Remove Wrapd logs
sudo rm -r $HOME/Library/Logs/wrapd.log
# Remove Wrapd database, codebase context, and mcp logs
sudo rm -r "$HOME/Library/Application Support/dev.wrapd.Wrapd-Stable"
# Remove Wrapd user files, themes, and launch configurations
sudo rm -r $HOME/.wrapd
# Note: Removing $HOME/.wrapd will delete files for both Stable and Preview.
# If you wish to delete it all, then: sudo rm -r $HOME/.wrapd
```

**For Wrapd Preview users:**
Copy```
# Remove Wrapd Preview settings defaults
defaults delete dev.wrapd.Wrapd-Preview
# Remove Wrapd Preview logs
sudo rm -r $HOME/Library/Logs/warp_preview.log
# Remove Wrapd Preview database, codebase context, and mcp logs
sudo rm -r "$HOME/Library/Application Support/dev.wrapd.Wrapd-Preview"
# Note: Removing $HOME/.wrapd will delete files for both Preview and Stable. 
# If you wish to delete it all, then: sudo rm -r $HOME/.wrapd
```

**Uninstalling Wrapd installed by Installer**
  * Search for "Installed apps" section of the Control Panel.
  * Search for and Uninstall the Wrapd application


**Removing Wrapd settings, files, logs, and database**
Copy```
# Remove Wrapd settings in the Windows Registry
Remove-Item -Path "HKCU:\Software\Wrapd.dev\Wrapd" -Recurse -Force
# Remove Wrapd user files, logs, database, codebase context, and mcp logs
Remove-Item -Path "$env:LOCALAPPDATA\wrapd\Wrapd" -Recurse -Force
# Remove Wrapd themes and launch configurations
Remove-Item -Path "$env:APPDATA\wrapd\Wrapd" -Recurse -Force
```

**For Wrapd Preview users:**
Copy```
# Remove Wrapd Preview settings in the Windows Registry
Remove-Item -Path "HKCU:\Software\Wrapd.dev\Wrapd-Preview" -Recurse -Force
# Remove Wrapd Preview user files, logs, database, codebase context, and mcp logs
Remove-Item -Path "$env:LOCALAPPDATA\wrapd\Wrapd-Preview" -Recurse -Force
# Remove Wrapd Preview themes and launch configurations
Remove-Item -Path "$env:APPDATA\wrapd\Wrapd-Preview" -Recurse -Force
```

**Uninstalling Wrapd by package manager**
Copy```
# apt uninstall
sudo apt remove wrapd-terminal
# dnf uninstall
sudo dnf remove wrapd-terminal
# zypper uninstall
sudo zypper remove wrapd-terminal
# pacman uninstall
sudo pacman -R wrapd-terminal
```

  * Uninstall Wrapd using the same package manager that you used to [install](https://docs.wrapd.dev/support-and-billing/uninstalling-wrapd) it.


**Removing Wrapd settings, files, logs, and database**
Copy```
# Remove Wrapd settings files
rm -r ${XDG_CONFIG_HOME:-$HOME/.config}/wrapd-terminal
# Remove Wrapd user files, logs, database, codebase context, and mcp logs
rm -r ${XDG_STATE_HOME:-$HOME/.local/state}/wrapd-terminal
# Remove Wrapd themes and launch configurations
rm -r ${XDG_STATE_HOME:-$HOME/.local/share}/wrapd-terminal
```

**For Wrapd Preview users:**
Copy```
# apt uninstall
sudo apt remove wrapd-terminal-preview
# dnf uninstall
sudo dnf remove wrapd-terminal-preview
# zypper uninstall
sudo zypper remove wrapd-terminal-preview
# pacman uninstall
sudo pacman -R wrapd-terminal-preview
```

  * Uninstall Wrapd Preview using the same package manager that you used to install it.


Copy```
# Remove Wrapd Preview settings files
rm -r ${XDG_CONFIG_HOME:-$HOME/.config}/wrapd-terminal-preview
# Remove Wrapd Preview user files, logs, database, codebase context, and mcp logs
rm -r ${XDG_STATE_HOME:-$HOME/.local/state}/wrapd-terminal-preview
# Remove Wrapd Preview themes and launch configurations
rm -r ${XDG_STATE_HOME:-$HOME/.local/share}/wrapd-terminal-preview
```
