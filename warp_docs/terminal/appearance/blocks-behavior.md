# Blocks Behavior
Wrapd lets you customize your Blocks in a variety of ways such as enabling Compact mode, or disabling the Block Dividers for a more custom block experience, and more.
## 
[](https://docs.wrapd.dev/terminal/appearance/blocks-behavior#compact-mode)
Compact Mode
Wrapd offers the option to enable Compact mode, which condenses the spacing between [Blocks](https://docs.wrapd.dev/terminal/blocks), enabling more content to be in view.
### 
[](https://docs.wrapd.dev/terminal/appearance/blocks-behavior#how-to-enable-compact-mode)
How to enable Compact Mode
Compact mode is disabled by default, but can be toggled in the following ways:
  * Navigate to `Settings > Appearance > Blocks > Compact Mode`.
  * Utilize the [Command Palette](https://docs.wrapd.dev/terminal/command-palette), then search for "Compact mode" to toggle.


Wrapd will open with the same compact settings in future sessions.

## 
[](https://docs.wrapd.dev/terminal/appearance/blocks-behavior#block-dividers)
Block Dividers
Wrapd [Blocks](https://docs.wrapd.dev/terminal/blocks) are divided by horizontal lines that separate individual command input and output, they create a visual break between different commands that you run in a session.
### 
[](https://docs.wrapd.dev/terminal/appearance/blocks-behavior#how-to-toggle-block-dividers)
How to toggle Block Dividers
Block dividers are enabled by default, but can be toggled in the following ways:
  * Navigate to `Settings > Appearance > Blocks > Show block dividers`.
  * Utilize the [Command Palette](https://docs.wrapd.dev/terminal/command-palette), then search for "Block Dividers".
